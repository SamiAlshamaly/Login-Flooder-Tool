## ✨ الميزات
- **واجهة رسومية (GUI)** سهلة الاستخدام.
- **إرسال طلبات متعددة** بخيارات متقدمة:
  - دعم **Tor** لتغيير عنوان IP.
  - توليد بيانات عشوائية (يوزرنيم/باسوورد).
  - تناوب وكلاء المستخدم (User-Agent).
- **سجل تفصيلي** للطلبات الناجحة والفاشلة.
